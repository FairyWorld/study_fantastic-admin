// status: true / false
proxy.$mainPageMaximize(status)

this.$mainPageMaximize(status)

# 第三方库

除 vue-router 、pinia 、axios 之外，框架还默认集成了一些常用且成熟的类库或插件，熟悉并熟练使用它们，能让你在开发过程中更得心应手。

# dayjsopen in new window

轻量的处理时间和日期的库。

const { proxy } = getCurrentInstance()
proxy.$dayjs

this.$dayjs

# vue-cookiesopen in new window

cookies 操作库。

const { proxy } = getCurrentInstance()
proxy.$cookies

this.$cookies

# HotKey.jsopen in new window

快捷键监听的库。

const { proxy } = getCurrentInstance()
proxy.$hotkeys

this.$hotkeys

# vue3-clipboardopen in new window 专业版

剪贴板的库，可将数据拷贝到系统剪贴板中。

const { proxy } = getCurrentInstance()
proxy.$clipboard

this.$clipboard

# QRCodeopen in new window 专业版

生成二维码的库，可生成图片或 canvas 二维码。

const { proxy } = getCurrentInstance()
proxy.$qrcode

this.$qrcode

# mittopen in new window

全局事件总线的库，也是 Vue 3 官方推荐的。

const { proxy } = getCurrentInstance()
proxy.$eventBus

this.$eventBus

# iNotifyopen in new window 专业版

支持 favicon、网页标题、Chrome桌面通知。

const { proxy } = getCurrentInstance()
proxy.$iNotify

this.$iNotify
